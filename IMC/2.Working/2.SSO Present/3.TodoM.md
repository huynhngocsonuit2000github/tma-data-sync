# SSO for user
* Create user at Default Directory

* Create the App roles at the app registrations

* Create Users and group
	- add user from Directory
	- add app role to map them
	-> then that user can access to that application

* Go to Enterprise Application to add the Permission
  - Go to TodoWokerApp
  - Add permission, that is the TodoApp role
